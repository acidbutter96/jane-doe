CSS3

As folhas de estilo servem para estilizar o nosso conteúdo HTML. Como o CSS, podemos modificar as cores, o background, as tipografias, larguras, alturas, etc. Tambpen podemos gerar animações e transições. Contamos com 3 métodos para vincular nossos arquivps CSS com o documento HTML: Externo e Interno



Seletores de CSS



Temos tres tipos de seletores:

  • Seletor de ID
    • Só posso usar uma vez na estrutura
    • Como CPF
    • Selecionado na class como #
  • Seletor de Classe
    • Pode ser aplicado em várias classes
    • Selecionados no CSS com um ponto
    <
  • Seletor de tag
    • Referenciado como o nome do método


  • Seletores combinados
    • h2.subtitulo

  • Subtítulo

  • Primeiro item



O CSS sempre vai priorizar os seletores mais específicos para aplicar os estilos
